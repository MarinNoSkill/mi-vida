# Carta de cumpleaños — Emiliana

Pequeña app React (Vite) que muestra una carta con fondo de corazones y un collage de 6 fotos.

Probado para ejecución local y despliegue en Vercel.

Rápido inicio (Windows, `cmd.exe`):

```
cd "c:\\Users\\camil\\Downloads\\mi vida\\mi-vida-card"
npm install
npm run dev
```

Construir para producción:

```
npm run build
npm run preview
```

Desplegar en Vercel (si tienes la CLI instalada):

```
vercel login
vercel --prod
```

Notas:
- Cambia las imágenes en `src/App.jsx` por tus propias URLs o coloca archivos en `public/` y apunta a ellos.
- `vercel.json` ya incluye configuración básica para builds estáticos.
